# Gold Price Monitoring

A modern end-to-end data pipeline project using:

- Scraper
- MongoDB
- dlt
- Supabase (Postgres)
- dbt
- Dagster
- Excalidraw

## High-level architecture

Website → Scraper → MongoDB → dlt → Supabase → dbt → Charts

Dagster orchestrates the full pipeline.
